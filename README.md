# minimal-task-tracker

A simple todo application for managing tasks. The interface should be clean and minimalistic, with the ability to add, remove, and mark tasks as completed. Preferably, use a light color scheme with a focus on usability and accessibility.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/minimal-task-tracker.git
cd minimal-task-tracker
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
